Available Tools

  • conversations_history - Get messages from channels and DMs
  • conversations_replies - Get thread messages
  • conversations_add_message - Post messages to channels
  • conversations_search_messages - Search messages with filters
  • channels_list - List all channels

Available Resources

  • slack://workspace/channels - Directory of all channels
  • slack://workspace/users - Directory of all users

MCP Endpoint

The MCP server is available at /mcp/ endpoint. Use this URL with your MCP client to connect to the Slack MCP server.

Authentication

Make sure to set the following environment variables:

  • SLACK_MCP_XOXC_TOKEN - Your Slack xoxc token
  • SLACK_MCP_XOXD_TOKEN - Your Slack xoxd token

Or alternatively use:

  • SLACK_MCP_XOXP_TOKEN - Your Slack OAuth token
